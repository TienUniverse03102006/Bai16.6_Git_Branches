nhánh Main

  1. Xóa lịch sử trên terminal: clear
  2. Kiểm tra nhánh: git branch or git branch -a
  3. Tạo nhánh mới trên máy tính: git branch -c tên nhánh
  4. Để chuyển sang nhánh mới: git checkout tên nhánh
  5. Muốn tạo nhánh mới và chuyển sang nhánh mới luôn: git checkout -b tên nhánh mới
  6. Khi chuyển nhánh các code ở trong từng nhánh không ảnh hưởng lẫn nhau
  7. Đẩy nhánh lên github: git push -u origin tên nhánh
  8. Lần đầu push lên thì sử dụng đầy đủ git push -u hoặc git push --set-upstream, từ lần 2 trở đi chỉ cần dùng git push
